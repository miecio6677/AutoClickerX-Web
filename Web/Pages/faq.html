FAQ

Q: How do I open the ClickGUI?

A: The default ClickGUI key is Right Shift (⇧). To change this, right-click the ClickGUI module and set a bind, or use: .bind add clickgui rshift.

Q: How do I use commands?

A: The default chat prefix is (*). Use this symbol before any command. For a full command list type *.help.

Q: How do I use Baritone?

A: Baritone uses (#) as its default prefix. For more details check the official Baritone documentation.

Q: What is a HWID? / How do I get a HWID reset?

A: AutoclickerX binds your license to your hardware ID. After changing parts or switching PC, request a reset via support ticket or email supportACX@gmail.com.

Q: The game is crashing, how do I fix this?

A: Send your crash log to supportACX@gmail.com.

Q: How do I suggest a new feature or report a bug?

A: Open an issue on the GitHub issue tracker.
